# JavaScript / TypeScript Developer Guidelines

## Modern Standards
- **ES6+:** Use modern syntax (const/let, arrow functions, destructuring).
- **Async/Await:** Prefer `async/await` over raw Promises/callbacks.
- **TypeScript:** If using TS, avoid `any`. Define interfaces/types explicitly.

## Ecosystem
- **Formatting:** Use Prettier.
- **Linting:** Respect ESLint rules.

## Testing
- **Jest/Vitest:** Use Jest or Vitest for unit testing.
- **Async Testing:** Ensure async tests are properly awaited.
