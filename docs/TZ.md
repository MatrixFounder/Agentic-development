# ТЗ: Переход на лицензию Apache 2.0

## 1. Общее описание
Задача: Сменить лицензию проекта с MIT на Apache License 2.0.
Цель: Обеспечить правовую защиту и условия использования, соответствующие Apache 2.0.

## 2. Список юзер-кейсов

### UC-01: Просмотр лицензии
**Актёры:** Разработчик, Пользователь
**Предусловия:** Файл LICENSE существует.
**Основной сценарий:**
1. Пользователь открывает файл LICENSE.
2. Пользователь видит текст Apache License 2.0.

**Критерии приёмки:**
- ✅ Файл LICENSE содержит полный текст Apache License 2.0.
- ✅ Copyright указан как "2025 MatrixFounder".

## 3. Ограничения
- Текст лицензии должен быть стандартным (без модификаций, кроме Copyright).
