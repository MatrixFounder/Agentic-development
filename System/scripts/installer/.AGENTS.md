# `installer/` — Framework Installer Package

Bootstrap-time tool that deploys the agentic-development framework into a
target project under a chosen vendor profile. **Not** part of the runtime
agent pipeline. Entry point: [../install.py](../install.py); bash wrapper:
[../../../install.sh](../../../install.sh). Spec: [docs/TASK.md](../../../docs/TASK.md)
(Task 063); design: [docs/ARCHITECTURE.md §9](../../../docs/ARCHITECTURE.md).

## Files

| File | Responsibility |
|------|----------------|
| `cli.py` | Subcommand dispatch — `main()` routes to `install`/`switch`/`update`/`uninstall`/`doctor`. |
| `vendors.py` | `vendors.yaml` loader + per-action schema validator. |
| `state.py` | Read/write `<target>/.agentic-installer-state.json`. |
| `framework_root.py` | Create/validate `target/.agentic-development/` (symlink or copy). |
| `symlinks.py` | `link_one` / `link_per_item` / `link_folder` / `make_dir` actions. |
| `copy.py` | `shutil.copytree` wrapper with the framework ignore-list. |
| `managed_block.py` | Marker-delimited managed block + SHA-256 anti-clobber (shared by gitignore + bootstrap). |
| `bootstrap.py` | Vendor bootstrap strategies: `at_import` / `marker_block` / `none`. |
| `gitignore.py` | `.gitignore` managed block + project-local `!`-exception scanner. |
| `backup.py` | Timestamped snapshots + retention. |
| `platform.py` | Windows detection + symlink-capability probe. |
| `errors.py` | `InstallerError` hierarchy with exit codes 1–4. |

Config: [../vendors.yaml](../vendors.yaml) — 5 vendor profiles + `defaults`.
Tests: [tests/installer/](../../../tests/installer/) (`unittest`).

## Conventions

- Dependencies: Python stdlib + `PyYAML` only (NFR-5).
- Built Stub-First: Task 063-01 created every module skeleton; Tasks
  063-02…063-10 fill logic layer by layer; Task 063-11 adds integration tests.
